Webbutvecklare .NET

Jag bygger användarvänliga webbapplikationer med fokus på modern frontend, tillgänglighet och tydlig struktur. Just nu studerar jag till Webbutvecklare .NET och specialiserar mig på att skapa hållbara digitala lösningar.

Om mig

Efter flera terminer inom medicinska studier har jag landat rätt i webbutveckling. Jag tar med mig analytisk förmåga och ett tydligt problemlösningsfokus. Som person är jag engagerad och nyfiken och trivs när jag får fördjupa mig, förstå helheten och bidra med strukturerade lösningar.

Tekniska färdigheter

Jag arbetar inom .NET-ekosystemet och bygger tydlig, strukturerad och användarvänlig kod. Min kompetens sträcker sig från backendlogik i C# till grundläggande frontendutveckling.

  • C# & .NET

    Stark grund i objektorienterad programmering, applikationsstruktur och moderna .NET-arbetsflöden.

  • HTML & CSS

    Skapar responsiva layouts med fokus på tydlighet, semantik och tillgänglighet (WCAG).

  • Javascript (grund)

    Används för enklare logik, validering och interaktivitet i frontend.

  • Git & GitHub

    Arbetar strukturerat med branching, commits, pull requests och versionshantering i team.

  • xUnit

    Skriver enhetstester för att säkerställa funktionalitet och förebygga regressionsfel.

Metodik & design

Jag arbetar analytiskt och metodiskt, med fokus på struktur, förståelse och långsiktig kvalitet. Min styrka ligger i att se både helhet och detaljer i systemdesign.

  • Objektorienterad analys och design

    Arbetar med att modellera problem, identifiera objekt, relationer och systemstruktur.

  • UML

    Erfarenhet av use case-, klass- och sekvensdiagram för att visualisera flöden och systemlogik.

  • Designmönster

    Grundläggande förståelse för vanliga mönster såsom Singleton och Repository.

  • Kravanalys

    Bryter ner funktionella behov, analyserar problem och dokumenterar lösningar strukturerat.
