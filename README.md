# Webbportfölj

Personlig webbportfölj utvecklad inom utbildningen Webbutvecklare .NET (Nackademin).

Syftet är att presentera mig som utvecklare, visa tekniska kunskaper och projekt samt demonstrera arbete enligt moderna frontend-principer med fokus på tillgänglighet, struktur och användarupplevelse.

Portföljen innehåller både färdigställda projekt och projekt under vidareutveckling. GitHub-länkar finns till respektive repository och live-demos publiceras där det är tillämpligt.

## Innehåll
Webbportföljen innehåller följande sektioner:
- Header med logotyp och navigering
- Responsiv navigation (hamburger‑meny för mobil)
- Sidebar med profil, kontaktuppgifter, sociala länkar och väderinformation (API)
- Hero‑sektion
- Om mig
- Kunskaper (teknik, metodik och design)
- Projektsektion (GitHub + live demo) 
- Referenser 
- Kontaktformulär med validering
- Footer

## Tekniker och verktyg
### Frontend
- HTML5 – semantisk struktur
- CSS3 – component based CSS, Flexbox, Grid, CSS‑variabler
- JavaScript (ES6) – interaktivitet, menylogik och formulärvalidering

### Övrigt
- Git & GitHub – versionshantering med feature branches
- VS Code – utvecklingsmiljö
- Font Awesome & SVG‑ikoner
- Google Fonts
- OpenWeather API – extern API‑integration


## Tillgänglighet (WCAG 2.1 AA)
Webbplatsen är utvecklad med fokus på WCAG 2.1 nivå AA och POUR‑principerna.
Exempel:
- Semantisk HTML och korrekt rubrikhierarki
- Tangentbordsnavigation
- Tydliga fokusmarkeringar (:focus-visible)
- ARIA‑attribut där semantik inte räcker (ex. mobilmeny)
- Formulär med labels, felmeddelanden och aria-live
- Färg används aldrig som enda informationsbärare
- Responsiv layout som fungerar vid zoom och textförstoring 

En separat WCAG‑rapport finns som PDF.


## Responsiv design
WWebbplatsen är anpassad för:
- Mobil
- Desktop
  
Layouten bygger på flexibla containers utan fasta höjder för att säkerställa god användbarhet vid olika skärmstorlekar och textförstoring.
