Webbportfölj

Detta projekt är en personlig webbportfölj skapad som del av utbildningen Webbutvecklare .NET (Nackademin). Syftet med webbplatsen är att presentera mig som utvecklare, visa mina tekniska kunskaper och projekt samt demonstrera arbete enligt moderna frontend‑principer med fokus på tillgänglighet, struktur och användarupplevelse.
Portföljen innehåller projekt som är under utveckling. GitHub-länkar finns till färdig kod, medan live-demos kommer att publiceras när projekten är klara.

Innehåll
Webbportföljen innehåller följande sektioner:
- Header med logotyp och navigering
- Responsiv navigation med hamburger‑meny för mobil
- Sidebar med profil, kontaktuppgifter, sociala länkar och väderinformation (API)
- Hero‑sektion
- Om mig
- Kunskaper (teknik, metodik och design)
- Portfölj med projekt under utveckling (GitHub + kommande live demo)
- Referenser 
- Kontaktformulär med validering
- Footer


Tekniker och verktyg
Frontend
- HTML5 – semantisk struktur
- CSS3 – component based CSS, Flexbox, Grid, CSS‑variabler
- JavaScript (ES6) – interaktivitet, validering och menylogik
Övrigt
- Git & GitHub – versionshantering, feature branches och commits

- Font Awesome & SVG‑ikoner
- Google Fonts
- OpenWeather API – extern API‑integration


Tillgänglighet (WCAG)
Webbplatsen är utvecklad med fokus på WCAG 2.1 nivå AA och POUR‑principerna.
Exempel på tillämpade riktlinjer:
- Semantisk HTML och korrekt rubrikhierarki
- Stöd för tangentbordsnavigation
- Tydliga fokusmarkeringar (:focus-visible)
- ARIA‑attribut där semantik inte räcker (t.ex. mobilmeny)
- Formulär med labels, felmeddelanden och aria-live
- Färg används aldrig som enda informationsbärare
- Responsiv layout som fungerar vid textförstoring och zoom

En separat WCAG‑rapport finns som PDF.
